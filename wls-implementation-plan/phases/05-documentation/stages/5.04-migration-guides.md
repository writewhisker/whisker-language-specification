# Migrating from Twine to WLS

## Overview

WLS can import stories from Twine (Harlowe and SugarCube formats).
This guide covers manual conversion and the automated import tool.

## Quick Reference

### Harlowe to WLS

| Harlowe | WLS |
|---------|-----|
| `(set: $var to value)` | `{do $var = value}` |
| `(if: cond)[content]` | `{cond}\\ncontent\\n{/}` |
| `(else:)[content]` | `{else}\\ncontent\\n{/}` |
| `[[text\|target]]` | `+ [text] -> target` |
| `[[target]]` | `+ [target] -> target` |
| `(print: $var)` | `$var` |
| `(goto: "passage")` | `-> passage` |

### SugarCube to WLS

| SugarCube | WLS |
|-----------|-----|
| `<<set $var to value>>` | `{do $var = value}` |
| `<<if cond>>...<</if>>` | `{cond}...{/}` |
| `<<elseif cond>>` | `{elif cond}` |
| `[[text\|target]]` | `+ [text] -> target` |
| `<<print $var>>` | `$var` |

## Automated Import

```bash
whisker import story.html --from=twine -o story.ws
```
