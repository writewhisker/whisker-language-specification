# Stage 5.05: Examples Library

## Objective
Create comprehensive examples library.

## Files to Create
- `examples/basic/*.ws`
- `examples/intermediate/*.ws`
- `examples/advanced/*.ws`
- `examples/showcase/*/`

## Prompt

```
Create examples library:

1. Basic examples (examples/basic/):

hello-world.ws:
```whisker
:: Start
Hello, world!
```

simple-choices.ws:
```whisker
:: Start
What do you want to do?
+ [Go left] -> Left
+ [Go right] -> Right

:: Left
You went left.

:: Right
You went right.
```

variables.ws:
```whisker
:: Start
{do $name = "Player"}
{do $score = 0}
Hello, $name! Your score is $score.
```

[10+ basic examples]

2. Intermediate examples:

inventory-system.ws:
```whisker
{do $inventory = []}

:: Start
You have {#$inventory} items.
+ [Pick up key] {do $inventory.push("key")} -> Start
+ [Check inventory] -> ShowInventory

:: ShowInventory
Your items:
{for $item in $inventory}
- $item
{/for}
+ [Back] -> Start
```

dialogue-tree.ws, branching-story.ws, etc.

3. Advanced examples:

state-machine.ws, procedural-dungeon.ws, etc.

4. Showcase (complete games):

examples/showcase/
├── mystery/          # Murder mystery game
│   ├── story.ws
│   └── README.md
├── adventure/        # Text adventure
└── visual-novel/     # Conversation-heavy

Each example:
- Well-commented
- Demonstrates specific feature
- Runnable standalone
```

## Expected Output
30+ example files.

## Verification
```bash
find examples -name "*.ws" | wc -l
# Should be 30+
```
